# Internal Rate of Return (IRR)

**Course:** Financial Accounting
**Source:** Session 05.pdf

## Definition

The discount rate that makes the net present value (NPV) of all cash flows from a particular project equal to zero.

## Key Points

- Used to evaluate the profitability of potential investments.
- Invest if the IRR exceeds the required rate of return for the risk level.
- Calculated using the cash flow sequence of an investment.

## Formulas & Equations

- 0 = Σ (Cash_Flow_t / (1 + IRR)^t)

## Examples

- Calculating the yield on a government bond.
- Evaluating an investment of $350k for a 4-year return stream.
