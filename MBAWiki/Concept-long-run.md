# Long Run

**Course:** Microeconomics
**Source:** Slides Week 4 Weekend-1.pdf

## Definition

A time horizon where all factors of production are variable, allowing firms to adjust their scale of operations fully.

## Key Points

- All inputs of production can be changed
- Used to assess returns to scale of the production function
- Time required to reach this state varies by industry
