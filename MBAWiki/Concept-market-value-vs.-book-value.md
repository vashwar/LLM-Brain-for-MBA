# Market Value vs. Book Value

**Course:** Financial Accounting
**Source:** Session 01.pdf

## Definition

The distinction between the book value of a firm (Shareholders' Equity on the balance sheet) and its market capitalization (share price multiplied by total shares).

## Key Points

- Book value is based on historical accounting costs.
- Market value reflects investor expectations of future prospects.
- The Market-to-Book ratio is a key metric for valuation comparison.

## Formulas & Equations

- Market Value = Share Price * Number of Shares
- Market-to-Book Ratio = Market Value / Book Value

## Examples

- Comparing Netflix's book value ($20.6B) to its market cap ($202.8B) resulting in a ratio of 9.84.
