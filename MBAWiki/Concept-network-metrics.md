# Network Metrics

**Course:** Leading People
**Source:** LP_2024_class_11_WE.pdf

## Definition

Quantitative measures used to identify an individual's position and influence within a social network structure.

## Key Points

- Betweenness: Measures the extent to which an individual acts as a 'bridge' between others who are not directly connected.
- Eigenvector Centrality: Weights an individual's connections by how well-connected those connections are (connecting to 'influentials').
- Constraint: Measures the extent to which an individual's connections are also connected to each other; lower constraint implies more diverse knowledge access.
