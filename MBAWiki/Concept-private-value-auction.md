# Private Value Auction and Common Value Auction

**Course:** Microeconomics
**Source:** Slides Week 3 - with notes.pdf

## Definition

*   **Private Value Auction:** An auction format where each bidder has a unique, independent valuation for the item, which is known only to the bidder.
*   **Common Value Auction:** An auction where the item has an identical underlying value to all bidders, but that value is unknown at the time of bidding.

## Key Points

- In private value auctions, bidder valuations are private and not necessarily known by others.
- Each bidder must form expectations about other bidders' valuations and strategies.
- Surplus is the difference between valuation and the bid.
- In common value auctions, bidders must estimate the true value based on information available.
- The winner of a common value auction is prone to the [[Winners Curse]] if their estimate is overly optimistic.
- Everyone would agree on the value of a common value item if it were known.
- Information asymmetry regarding the true value is the primary driver of bidder behavior in common value auctions.

## Formulas & Equations

- Expected Surplus = Prob(Bid wins) * (Valuation - Bid)

## Examples

- Private Value: Picasso painting, a meal at a restaurant.
- Common Value: Oil field lease, a jar full of an unknown number of coins.

## Notes

There is a fundamental distinction between private and common value auctions regarding strategic bidding and the role of information.
