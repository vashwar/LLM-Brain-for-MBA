# Returns to Scale

**Course:** Microeconomics
**Source:** Slides Week 4 Weekend-1.pdf, Fall 2024 - EWMBA 201A-LEC-31A _ 32A - Economics for Business Decision Making_Captions_English (United States).txt

## Definition

The rate at which output increases as all inputs are increased proportionately; the relationship between a proportional increase in all inputs and the resulting change in output in the long run.

## Key Points

- Increasing returns to scale: output increases more than proportionately.
- Constant returns to scale: output increases exactly proportionally.
- Decreasing returns to scale: output increases less than proportionately.
- Distinct from diminishing returns, which applies when only one factor is increased while others are held constant.

## Formulas & Equations

- Increasing: f(αL, αK) > αf(L, K)
- Constant: f(αL, αK) = αf(L, K)
- Decreasing: f(αL, αK) < αf(L, K)

## Examples

- Airline industry scaling from small planes to large planes.
- Cobb-Douglas function where exponents sum to 1 indicates constant returns to scale.

## Notes

Distinct from 'economies of scale', which relates to costs rather than production quantities. Returns to scale describes production technology.
