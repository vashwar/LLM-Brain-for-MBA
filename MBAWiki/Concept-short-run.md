# Short Run

**Course:** Microeconomics
**Source:** Slides Week 4 Weekend-1.pdf

## Definition

A time horizon in production during which at least one input factor is fixed and cannot be changed.

## Key Points

- Defined by context-specific constraints on capital or labor
- Used for calculating marginal product and marginal cost
- Distinction from long run depends on the time required to adjust all inputs
