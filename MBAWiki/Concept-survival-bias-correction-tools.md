# Survival Bias Correction Tools

**Course:** Leading People
**Source:** SelectionBiasAndThePerilsOfBenchmarking.pdf

## Definition

Statistical methods used to estimate the characteristics of an entire population when only data from survivors is available.

## Key Points

- Economists like James Tobin and James Heckman developed tools to correct for selection/survivorship bias.
- These tools often assume a specific distribution (like a bell curve) to infer the missing 'bottom' of the data set.
- Correcting for bias allows managers to obtain unbiased estimates of how investments (like training) influence performance.

## Examples

- Estimating average Return on Equity (ROE) for an industry by assuming a standard bell curve distribution even if low-profit businesses failed and are missing from the data.
- Using the Heckman correction to improve the cost-effectiveness of marketing investments.
